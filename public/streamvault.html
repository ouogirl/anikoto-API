📋 One-Time Setup (host this page on your API)

For downloads to work, this page needs to run on your Vercel domain. It takes 2 minutes:

  1. Go to your GitHub fork of anikoto-API
  2. Open the public/ folder
  3. Click "Add file" → "Create new file"
  4. Name it streamvault.html
  5. Paste this entire HTML page's source code (Ctrl+U to view, Ctrl+A to select all)
  6. Click "Commit changes"
  7. Wait ~60s for Vercel to auto-deploy
  8. Open https://your-app.vercel.app/streamvault.html
